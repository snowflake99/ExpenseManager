# Expense Manager

Keeping a balanced budget can help you stay on track and sleep better at night. But if your money-management system requires sifting through piles of receipts and retrieving cash from various pockets and purses, itâ€™s time to reconsider your approach. With Expense Manager, you can put away the paperwork and get a better understanding of your daily, weekly, monthly and annual spending habits. Expense Manager is a simple web based interface to manage all your expenses online on a LAMP stack.

# Features

1. User authentication. 
2. Addition and deletion of expense.
3. Rearrangement of entries using keyboard shortcut.
4. Upload and download records in CSV format. 
5. Expense summary in month and year views.
6. Administrator view to manage user account and tables.
